# 世界观设定

## 世界一句话
宗门联盟统治的大陆级高武世界，异界结晶是核心资源，异界入侵威胁人类存亡。

## 题材融合点（复合题材可填）
- 主题材：系统流
- 副题材：赛博朋克 + 高武 + 拯救世界
- 融合机制：金铲铲复利系统与宗门资源争夺深度绑定；赛博朋克体现在械丹/灵械/意识上传等修炼辅助科技

## 世界结构
- 大陆/位面数量：大陆级（多宗门林立）
- 核心区域：玄天域（宗门联盟总部所在地）
- 边缘区域：蛮荒域（异界入侵的前线）

## 势力格局
- 核心势力：宗门联盟（以青云宗为首的七大宗门）
- 次级势力：世俗凡人城市（依附宗门，上交异界结晶税）
- 敌对/中立关系：异界势力（入侵者）、散修联盟（中立）
- 宗门/组织层级：外门弟子→内门弟子→真传弟子→长老→掌门

## 社会结构
- 社会阶层：外门弟子（底层）→ 内门弟子 → 真传弟子 → 长老 → 掌门（顶层）
- 资源分配规则：异界结晶是核心硬通货，宗门贡献点是日常货币；境界越高，可获得的修炼资源越多
- 信仰/意识形态：实力为尊，宗门荣誉高于一切

## 历史年表（关键节点）
- 仙历3021年：宗门联盟成立，七大宗门共治
- 仙历3050年：第一次异界入侵，被击退，签订临时停战协议
- 仙历3080年：异界先头部队再次出现，冲突加剧

## 核心规则
- 资源稀缺性：异界结晶总量有限，整个大陆宗门弟子都在争夺
- 政治/宗门规则：宗门贡献点用于兑换修炼资源、功法、情报
- 社会常识/禁忌：外门弟子禁止进入内门修炼区；弟子间不得私斗（特殊情况需申请宗门仲裁）
- 硬约束（不可违背）：
  1. 异界结晶总量有限，主角成为移动靶子
  2. 系统铸造时产生神铸光芒，3秒内敌人可定位主角位置

## 货币与经济
- 货币体系：异界结晶（基础）→ 神铸结晶（高级）→ 宗门贡献点（日常）
- 兑换规则：1神铸结晶 = 10异界结晶；1异界结晶 = 100宗门贡献点
- 主要流通形态：电子积分（宗门内）、结晶碎片（黑市）
- 交易场景：宗门贡献阁（兑换）、拍卖行（高价）、黑市（稀有）

## 地理版图
- 重要地点清单：
  - 青云宗（主角宗门，七大宗门之首）
  - 宗门垃圾场（主角获得系统的地方）
  - 蛮荒域（异界入侵前线）
  - 宗门贡献阁（资源兑换中心）
- 关键资源点：异界结晶矿脉（宗门联盟控制）

## 日常生活
- 货币/交易方式：宗门贡献点积分制
- 交通/通信：传音符（短距离）、跨域传送阵（代价高）
- 教育/职业体系：宗门弟子修炼体系，贡献点决定可接触的功法等级
