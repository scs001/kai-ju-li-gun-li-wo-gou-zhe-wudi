# 力量体系设定

## 体系公理
- 核心信条/定律：实力为尊，境界决定一切；资源决定境界上限
- 代价规则：每一次境界突破都需要消耗大量异界结晶；突破失败可能造成永久性损伤
- 公平性原则（谁能破格、谁必须付出代价）：主角的金铲铲系统是唯一破格工具，但每次铸造都会暴露位置（代价等于优势）

## 体系类型
- 体系类型：高武 + 赛博朋克化（武者 + 机械改造 + 宗门功法）
- 典型境界链：淬体境 → 炼气境 → 筑基境 → 金丹境 → 元婴境 → 化神境 → 大乘境 → 渡劫境
- 小境界划分：每境9重（如：淬体一重 ~ 淬体九重）

## 能力来源
- 能量/资源来源：异界结晶（核心硬通货）；神铸结晶（高级，可加速修炼）
- 训练/修炼方法：宗门功法（心法）+ 械丹辅助（赛博朋克元素）+ 实战积累
- 社会控制机制：宗门贡献点制度；境界认证（需通过宗门考核）

## 等级体系
- 等级顺序：
  1. 淬体境（9重）：外门弟子基准
  2. 炼气境（9重）：内门弟子基准
  3. 筑基境（9重）：真传弟子基准
  4. 金丹境（9重）：长老基准
  5. 元婴境（9重）：副掌门基准
  6. 化神境（9重）：掌门基准
  7. 大乘境（9重）：宗门联盟长老
  8. 渡劫境（9重）：传说级，救世主级
- 每级核心能力：
  - 淬体境：肉身强化，力量/速度/防御基础
  - 炼气境：内劲外放，可使用初级功法
  - 筑基境：真气凝丹，可驾驭法器
  - 金丹境：金丹成就，寿命延长，功法威力倍增
  - 元婴境：元婴出窍，神识外放
  - 化神境：神魂化虚，法则感悟
  - 大乘境：接近法则本质
  - 渡劫境：渡劫飞升（本书最高境界）

## 晋级条件
- 资源要求：每突破需要消耗大量异界结晶；境界越高，消耗越大
- 突破方式：功法运转 + 资源辅助 + 心理关卡（心魔劫）
- 失败代价：轻则重伤，重则境界倒退，极端情况走火入魔
- 越级代价（越阶战斗/突破）：消耗加倍，需要特殊机缘

## 资源系统
- 核心资源类型：异界结晶（基础）、神铸结晶（高级）、宗门贡献点
- 资源获取方式：任务奖励、击杀异界生物、宗门兑换、黑市购买
- 稀缺性规则：异界结晶总量有限，大陆各宗门争夺激烈
- 替代路径/灰色交易：黑市走私、抢夺其他弟子

## 战斗规则
- 伤害与防御逻辑：境界压制为主，越级挑战极难；功法相克可弥补境界差距
- 战斗节奏特点：快节奏，胜负往往在数招内决定
- 克制/反制关系：金属性功法克木属性，火克金（但并非绝对）
- 逃生/撤退机制：保命符箓（一次性，可强制传送离开战场）

## 禁忌与限制
- 禁术/禁地：禁术"燃魂大法"（燃烧寿命换取短期力量）；禁地"异界裂缝"（高危区）
- 高阶限制：渡劫境不得对低境界弟子出手（宗门联盟铁律）
- 硬限制（不可绕过）：神铸光芒会暴露铸造者位置（主角金手指的代价）

## 体系漏洞与利用
- 漏洞（必须可解释）：金铲铲系统的利滚利机制（持10铸造1，收益再投入可产生复利）
- 主角如何利用：通过持续积累异界结晶，利用复利机制实现弯道超车
- 反派如何反制：抢夺主角的结晶、联合追杀、利用神铸光芒定位

## 反馈节奏
- 境界变化节点：每境突破需有明显数值/战力变化展示
- 读者可感知的战力差距展示方式：数值化展示（如：战力50→战力180）

## 与创意约束对齐
- 反套路规则如何体现：
  - AT-003（优势即致命弱点）：复利机制让主角积累越多，成为越大的靶子
  - GM-009（资源稀缺）：异界结晶总量有限，主角必须与全宗门争夺
- 硬约束如何绑定体系：
  - 铸造产生神铸光芒 → 主角无法偷偷变强，必须面对暴露风险
  - 谨慎导致错过最优解 → 主角需要在"等待积累"和"及时出手"间做抉择
