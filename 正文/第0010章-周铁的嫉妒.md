# 第10章 周铁的嫉妒

宗门大比倒计时，D-28，清晨。

天刚蒙蒙亮，垃圾场里就已经有了人影。

林尘比平时早到了半个时辰。他穿着一身灰色的外门弟子服，脸上带着惯常的漠然，像是例行公事一般往垃圾场深处走去。

他的步伐很慢，时不时地弯腰翻找着什么。

但他的余光一直在注意着不远处的动静。

果然。

周铁在跟踪他。

那道高大的身影躲在几堆杂物后面，远远地跟着。林尘每走一段路，他就会跟上来一段，始终保持着二十步左右的距离。

林尘的嘴角微微勾起。

上钩了。

昨天他故意在周铁能看到的地方"发现"了一块结晶，然后假装小心翼翼地藏起来。那块结晶其实只有指甲盖大小，根本不值得他那么紧张，但他偏偏表现得像发现了天大的宝贝。

周铁不是傻子，但他有一个致命的弱点——贪婪。

贪婪的人最容易上当，因为他们太想得到了，所以会忽略很多明显的破绽。

林尘继续往前走，时不时地停下脚步，假装在地上摸索。他的动作很自然，就像是一个普通弟子在找东西，但实际上，他一直在计算着距离和时机。

他已经观察过了，这个时间点，垃圾场深处很少有人来。如果周铁要动手，最好就是在他落单的时候。

所以他必须给周铁创造一个"完美的时机"。

走了大约一刻钟，林尘在一处偏僻的角落停下。

这里三面环抱着杂物堆，只有一个出口，正是设伏的绝佳地点。当然，这个"绝佳地点"是给周铁准备的。

他蹲下身子，假装在地上翻找着什么。

身后，周铁的脚步声越来越近。

"哟，林尘，这么早就来捡破烂了？"

声音从背后传来，带着几分戏谑。

林尘的心里一紧，但他的身体没有任何异常反应，只是慢慢站起身，转过头来。

"周师兄。"他的声音很平静，甚至带着几分恭敬，"这么早，您怎么也在这儿？"

周铁走过来，脸上的笑容里藏着刀："我听说这附近有个好地方，能捡到不少好东西。特意来看看。"

"是吗？"林尘的脸上露出一丝"茫然"，"我怎么不知道？"

周铁的眼睛眯了起来："你不知道？那昨天你在那边翻什么呢？"

林尘的表情微微一变，像是被戳中了什么。

就是这一瞬间的破绽，让周铁的心里有了底。

"算了，不跟你废话。"周铁的语气突然冷了下来，"把你手里的结晶都交出来。"

林尘的脸色变了："周师兄，我……我真的没有多少……"

"没有多少？"周铁冷笑一声，"你昨天藏起来的那块，当我瞎了？"

林尘往后退了一步，像是被吓到了："那是……那是我好不容易才找到的，周师兄您就放过我吧……"

他的声音里带着几分哀求，眼神也有些闪躲。

周铁的嘴角勾起一丝满意的弧度。

果然是个软柿子。

"少废话！"他往前走了一步，"我知道你有个秘密渠道，能稳定搞到结晶。乖乖交出来，我以后不再找你麻烦。否则……"

他没有说完，但威胁的意味已经很明显了。

林尘低着头，像是在犹豫。

他的脑子里飞快地转着。

周铁以为他有"秘密渠道"。这是好事。如果周铁一直这么认为，就会把注意力放在"渠道"上，而不是他本人。

但现在的问题是，周铁要他交出结晶。他手里只有七个碎片，如果全部交出去，第二次铸造就不知道要等到什么时候了。

不交。

但也不能硬顶。

他的脑子飞速运转着，忽然抬起头，眼神里带着几分"下定决心的意味"。

"周师兄，我可以给您一些，但不是全部。"

"嗯？"周铁挑了挑眉。

"我可以告诉您那个地方的规律。"林尘的声音压得很低，"每周三和周六的清晨，深处靠北的那个角落，会有新的结晶出现。这是我自己摸索出来的规律。"

周铁的眼睛亮了一下，但很快又恢复了冷静："你说的是真的？"

"千真万确。"林尘的演技堪称完美，"我之所以每次都能捡到结晶，就是因为这个。只是那个地方不太好找，我一开始也是碰巧发现的。"

周铁盯着他看了几秒，似乎在判断他话里的真假。

林尘的心里有些紧张，但他的表情没有任何破绽。

谨慎。谨慎。他告诉自己，谨慎是他在这个世界生存下去的唯一依仗。如果连这点都做不好，他早就不知道死了多少回了。

"好。"周铁终于开口，"那你先把这个星期的规律告诉我。"

林尘点点头，开始详细地描述起来。他的描述半真半假，把真正有用的信息藏在了一大堆无关紧要的细节里。周铁听得半信半疑，但贪婪最终还是战胜了理智。

"行了，我知道了。"周铁摆摆手，"这次的结晶你先留着，下次再敢私藏，别怪我不客气。"

说完，他转身离开了。

林尘站在原地，等周铁的身影完全消失之后，才缓缓松了一口气。

他的后背已经被冷汗浸透了。

好险。

周铁暂时被他糊弄过去了，但他知道，这只是暂时的。以周铁的性子，最多三五天，他一定会再来。到那时候，如果林尘不能拿出足够的结晶，情况只会更糟。

林尘握紧拳头。

三五天。

他必须在这三五天里，完成第二次铸造。

否则，等待他的将是真正的绝境。

他转身往垃圾场深处走去，脚步比来时快了很多。

周铁以为他得到了"秘密渠道"的信息，但实际上，林尘才是那个掌握信息差的人。

这就是他的优势。

谨慎让他多想了十步，而多想十步，就意味着他永远比对手看得更远。

哪怕只是一点点，也足够让他在这场博弈中占据先机。

林尘的嘴角微微勾起，目光落在远处某个不起眼的角落。

那里，藏着他这几天偷偷攒下的两块结晶碎片。加上现有的七个，三天后，他就能进行第二次铸造了。

周铁，等着吧。

他不是软柿子，也不是任人拿捏的废物。

他只是在等待时机。

而这个时机，很快就要到了。
