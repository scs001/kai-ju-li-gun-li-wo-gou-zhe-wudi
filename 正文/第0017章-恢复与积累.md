# 第17章 恢复与积累

林尘蜷缩在地窖的角落里，听着头顶上渐渐远去的脚步声，心跳依然如擂鼓。

太险了。

如果不是那个灵械残骸里残留的能量，他今晚可能就交代在深处区域了。

周铁。三个人。淬体境四重。

这个仇，他记下了。

林尘闭上眼睛，将意识探入面板深处。

【战力：90】

【精神力：55/100】

【异界结晶：0/10】

【神铸结晶：3】

【提示：持10个异界结晶可进行铸造】

战力确实提升了。从75到90，整整十五点的跨越。

可他的异界结晶归零了。

林尘的眉头微微皱起。

之前辛辛苦苦攒了八块结晶，加上地窖里捡的五块，一共十三块。结果一夜之间，全部用在了第二次铸造上。现在又回到了起点。

还得重新攒。

他深吸一口气，将烦躁的心情压下去。

急躁是没用的。这个世界就是这样，你方唱罢我登场，资源就那么多，你不抢就被别人抢走。

现在最重要的是恢复精神力。

55的精神力，连一半都不到。如果再遇到危险，他连逃跑的力气都没有。

林尘决定先休息。

他靠着地窖冰凉的石壁，慢慢闭上眼睛。地窖里很安静，只有偶尔从缝隙里传来的风声。干燥的沙土散发着陈旧的气息，让人有一种说不出的安心感。

不知道过了多久。

林尘是被一阵微弱的光线刺醒的。

他睁开眼睛，发现地窖入口处透进了一丝微弱的晨光。

清晨了。

他坐起身，活动了一下有些僵硬的筋骨。精神力恢复了不少，至少不再是那种随时会倒下的感觉。

林尘将意识探入面板。

【精神力：80/100】

不错。休息了一夜，精神力恢复到了八十点。

他轻手轻脚地爬出地窖，观察了一下四周的情况。废弃的械丹仓库周围很安静，没有人影。

林尘松了口气。

他沿着昨晚逃跑的路线，绕到了深处区域的边缘地带。动作很小心，尽量贴着阴影走。

深处区域的结晶确实更多。

不到一个时辰，他就找到了两块异界结晶。

加上之前在杂物堆里捡到的一块，一共三块。

还差七块。

林尘没有贪心。他知道见好就收的道理。周铁一定还在找他，如果待得太久，很可能会被发现。

他沿着原路返回，在另一处偏僻的角落又找到了两块。

五块了。

还差五块。

时间已经是下午。

林尘找了个隐蔽的地方坐下，将五块结晶收进怀里。这些结晶大小不一，大的有指甲盖大，小的只有米粒大。但都是货真价实的异界结晶，吸收之后可以转化为修炼资源。

他没有急着吸收。

现在最重要的是时间。D-26，距离宗门大比还有二十六天。他必须在那之前，尽可能地提升自己的实力。

林尘闭上眼睛，开始运转基础的修炼功法。

这是外门弟子都会的基础心法，不需要消耗任何资源，只需要时间和耐心。功法运转的过程中，周围的天地灵气会被缓慢吸收，滋养经脉和丹田。

虽然效率很低，但聊胜于无。

一个时辰后，林尘睁开眼睛。

【精神力：85/100】

【战力：90】

修炼让他的精神力又恢复了一些。距离满值不远了。

林尘站起身，活动了一下有些酸麻的腿脚。

他决定今晚再出去一趟。趁夜色掩护，深入深处区域再找几块结晶。

如果运气好的话，也许能凑够十块。

夜色降临。

林尘从藏身处钻出来，朝深处区域走去。

深处区域比外围更加危险。地形复杂，到处都是杂物和废弃的器械。一不小心就可能踩空或者触发某些机关。

但相应的，结晶也更多。

林尘的脚步很轻，目光在黑暗中扫视着。他的战力虽然只有九十，但感知能力已经比普通人强了不少。能听到五十丈外的脚步声，能看清黑暗中的一些细节。

这是战力提升带来的好处。

大约走了一刻钟，他在一堆废弃的器械残骸里发现了三块结晶。

三块。

运气不错。

林尘将结晶收进怀里，继续往前走。

又过了半个时辰，他在一处偏僻的角落里又找到了两块。

八块了。

还差两块。

林尘没有继续深入。他知道深处区域不是久留之地，万一遇到什么强大的敌人，他现在的实力根本不够看。

他开始往回走。

就在他穿过一片杂物堆的时候，前方突然传来一阵窃窃私语。

林尘的脚步猛地一顿，连忙屏住呼吸，躲到了一块废弃的铁板后面。

"……听说了吗？昨晚那道光……"

"嘘！小声点！"

林尘的心微微一沉。

昨晚的铸造……果然引起注意了。

他将耳朵贴近铁板，仔细听着那两人的对话。

"周师兄好像很生气，说一定要找到那个人……"

"可不是嘛。那道光那么大，整个垃圾场都看到了。周师兄说那肯定是什么宝贝……"

"能找到吗？"

"难说。那人跑得快，没准早就藏起来了。不过周师兄说了，谁要是能提供线索，赏五块异界结晶……"

林尘的瞳孔微微收缩。

五块异界结晶。

这个赏金不低。看来周铁是真的急眼了。

他悄悄后退了几步，然后沿着另一条路绕了过去。

那两个弟子的对话他听了个大概。看来昨晚的动静确实不小，连周铁都惊动了。不过他们似乎没有发现那道光和他有关，只是以为是什么宝贝出世。

这是个好事。

林尘加快了脚步，很快消失在夜色之中。

回到地窖的时候，已经是后半夜了。

他靠在墙上，大口大口地喘着粗气。

【异界结晶：8】

还差两块。

明天再找找，应该能凑齐。

林尘闭上眼睛，开始恢复精神力。

第二天一早，他趁着清晨人少的时候出去转了一圈。

运气不错，在一处偏僻的角落里捡到了两块被遗弃的结晶。

十块。

终于凑齐了。

林尘看着面板上的数字，心中涌起一股难以言喻的情绪。

十块异界结晶，又攒够了。

可以进行第二次……不对，是第三次铸造了。

林尘的嘴角微微勾起。

第三次铸造，不知道又会有什么惊喜。

战力从90提升到多少？

他很期待。

【战力：90】

【精神力：95/100】

【异界结晶：0/10】

【神铸结晶：3】

【提示：持10个异界结晶可进行铸造】

林尘盯着那个数字，心跳微微加快。

十块结晶，又攒够了。

第三次铸造。

他抬头看了看天色。太阳已经升到了正中，是一天中最明亮的时刻。

可他不能在这里铸造。神铸光芒太显眼了，整个垃圾场都能看到。必须找一个更隐蔽的地方。

林尘想起了深处区域的一处绝地。

那是一片被废弃的械丹仓库，地形复杂，很少有人去。如果在那里铸造，就算有光芒，也不会第一时间被发现。

他收拾好东西，朝深处区域走去。

第三次铸造，他要亲手完成。

战力提升，就在眼前。
